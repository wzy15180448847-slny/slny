2. 输入用户名: tenant0, 密码: 123456
3. 点击登录
4. **预期结果**: 成功登录，跳转到租客首页

### 2.2 房源搜索测试
1. 登录租客账号后进入首页
2. 在搜索框输入关键词如"精装"
3. 设置价格区间筛选（如 2000-5000）

5. **预期结果**: 显示符合条件的房源列表

1. 在房源列表中点击任意房源卡片
2. 进入房源详情页
3. 点击"预约看房"按钮
4. 选择预约时间，填写联系信息
5. 点击提交
6. **预期结果**: 预约提交成功，显示预约记录

### 2.4 查看预约记录
1. 点击顶部导航"我的预约"
2. **预期结果**: 显示所有预约记录，包含不同状态（待确认、已安排、已完成）

### 2.5 查看合同
1. 点击顶部导航"我的合同"
2. **预期结果**: 显示合同列表，可预览合同内容

### 2.6 查看账单
1. 点击顶部导航"我的账单"
2. **预期结果**: 显示账单列表，支持支付操作

### 2.7 提交报修
1. 点击顶部导航"报修管理"
2. 点击"提交报修"按钮
3. 填写报修内容，上传图片
4. 点击提交
5. **预期结果**: 报修提交成功

### 2.8 个人中心
1. 点击顶部导航"个人中心"
2. 可修改个人信息、查看钱包余额
3. **预期结果**: 页面正常显示，支持信息修改

## 三、房东角色冒烟测试

### 3.1 登录测试
1. 打开 http://localhost:5173/login
2. 输入用户名: landlord0, 密码: 123456
3. 点击登录
4. **预期结果**: 成功登录，跳转到房东首页

### 3.2 房源管理
1. 点击左侧导航"房源管理"
2. **预期结果**: 显示房东名下所有房源列表

### 3.3 发布房源
1. 点击"发布房源"按钮
2. 填写基础信息（标题、地址、面积等）
3. 上传房源图片（可使用默认图片）
4. 配置租金信息
5. 点击提交
6. **预期结果**: 房源提交成功，进入审核状态

### 3.4 处理预约
1. 点击左侧导航"预约管理"
2. 选择状态为"待确认"的预约
3. 点击"确认预约"或"拒绝预约"
4. **预期结果**: 预约状态更新成功

### 3.5 发起合同
1. 在预约列表中选择已完成的预约
2. 点击"发起合同"按钮
3. 填写合同内容
4. 点击发送
5. **预期结果**: 合同发送成功

### 3.6 账单管理
1. 点击左侧导航"账单管理"
2. 查看租客账单状态
3. 对逾期账单点击"发送催收通知"
4. **预期结果**: 通知发送成功

### 3.7 报修处理
1. 点击左侧导航"报修处理"
2. 选择待处理的报修单
3. 更新状态为"维修中"或"已修复"
4. **预期结果**: 报修状态更新成功
